## Getting Started

Install dependencies:
```bash
pnpm i
```

Start the development server:
```bash
pnpm run dev
```

Build for production:
```bash
pnpm run build
```

Về git:
- Local: Máy cá nhân
- Remote: Server của Git -> Github/ Gitlab/ Provider khác, v.v
- 2 giai đoạn
    - Commit: Đẩy những file thay đổi ở local lên server tạm
    - Push: Thực sự đẩy lên remote

- A -> B
- A -> C -> Pull từ remote về -> Conflict -> Resolve -> Push
